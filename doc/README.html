deployme Do you want to deploy your components automagically?

Usage: 

      *deployme* [ *--start* | *--stop* ]

      *deployme* [ *--report* | *--report pendings* | *--report currents* ]

      *deployme* [ *--report id=3DID* ]

      *deployme* [ *--commit id=3DID* | *--commit pendings* ]

      *deployme* [ *--rollback id=3DID* ]
