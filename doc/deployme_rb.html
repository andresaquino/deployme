Description

Synopsis 

  Do you want to deploy your components automagically?
  deployme was developed by Andres Aquino proudly sponsored with the time of his nextel's friends!

Examples

  Crear nuevo desplegado y dejar el archivo de configuracion listo para parametrizarlo
    deployme --add --workflow=NOE

  Revisar configuracion de un desplegado especifico
    deployme --check --workflow=NOE
  
  Mostrar los desplegados pendientes de autorizacion
    deployme --show --pending
  
  Mostrar todos los deplegados registrados
    deployme --show --all
  
  Mostrar todos los desplegados autorizados [default]
    deployme --show --authorized

  Mostrar todos los desplegados en proceso
    deployme --show --in-progress
  
  Ejecutar y dar flujo a los desplegados autorizados 
    deployme --commit --authorized
  
  Ejecutar y dar flujo a una orden de trabajo especifica
    deployme --commit --workflow=NOE | CCI | SisCob | etc...

  Ejecutar y dar flujo a todos los desplegados registrados
    deployme --commit --all

Usage 

  deployme [options] source_file

  For help use: deployme -h

Options

  --show              Show the workflows registerd
    --all             Default option, all workflows registered
    --pendings        Show all workflows pending to authorize
    --in-progress     Show all workflows in progress to deploy
  --commit            Execute deployment activity
    --all
    --pendings
  --rollback
    --last
    --workflow=NOE
  --stop 
    --last
    --all
    --in-progress
  --help              Displays help message
  --version           Display the version, then exit
  --quiet             Output as little as possible, overrides verbose
  --verbose           Verbose output
